转载-注册国外 AppleID

偶然看到一篇创建国外的AppleID的文章，之前也使用过别人的国外AppleID来下载app，就准备尝试一下创建自己的国外AppleID，亲测是可以创建的。

准备工作

  1. 一部 iphone 或者 ipad
  2. 一个能接受验证码的手机号（注册过国内 ID 的也可）

注册教程

  1. iPhone 或者 ipad 的设置退出原来的 AppleID，点击登录 -> 没有或忘记 Apple ID。注意，退出 ID 是在 设置里面，不是在 APP Store

  2. 选择 创建 Apple ID 输入姓名、出生日期等常规信息，然后 下一步（年龄要大于 18 岁）

  3. 这里我们选择使用电子邮箱地址 APP Store，再选择没有电子邮件地址 APP Store。这样就可以免费创建一个苹果 iCloud 电子邮件地址了，方便快捷。

  4. 选择获取 iCloud 电子邮件地址，接着在 “电子邮件” 框框里输入方便记忆的账号名。然后点击下一步（Apple 新闻和公告可以关掉）

  5. 接着是设置密码，这步很关键。密码必须要设置 8 位数以上，且要同时包括数字、大小写字母。比如 123abcABC 这样就是可以的。设置好后然后 下一步 。

重要提醒：密码里不要包括有刚设置的姓名、电话号码或电子邮件地址中的某个部分，否则会导致无法创建账户

  1. 接着验证电话号码。输入自己常用的号码、接收短信验证码填入后会自动进入下一步。

说个情况，如果是没有插卡的手机，填入电话号码下一步后它会提示“无法创建账户”

  1. 继续，并同意使用条款。

  2. 经过几十秒的等待，你的 Apple ID 就初步创建完成了。输入手机密码进入下一步, 如跳出提示是否合并，选择 “不合并”

  3. 如果不需要这个帐号同步你的帐号信息，建议把同步功能给关掉。

  4. 接着打开 APP Store，会让你完善 Apple ID 的相关信息。

    Ps：如没跳出 “检查” 提示框也没关系，点击帐号 — 输入密码并登录，也会跳出 “检查” 提示框。

  5. 地区更改为 “美国” 或者你想设置的其他地区。这里以注册美区为例，所以选择美国。更改好后同意使用条款，然后 “下一步”

  6. 这步很关键，首先付款方式不需要填写。然后因为我们要注册的是美区ID，所以账单寄送地址必须填美国的地址，这需要借助美国地址生成器（也可以自己在网上百度一个美国地址填上去）。

  7. ok，完成地址的验证，你的美区 ID 就算成功注册并可以正常使用了。APP Store 的界面也自动切换到了美区的，搜个国区没有的游戏试试… 完全没问题～

注意事项

  1. 填写地址时提示时，区号记得不要填 “1” 开头的；

  2. 后续使用帐号，最好就不要通过设置里去登录，直接在 APP Store 登录即可；

  3. 登录后如还是国区界面，后台退出 APP Store 后再打开即可
