chatGPT体验

注册OpenAI

注册地址：http://beta.openai.com/signup

注册之后发现OpenAI 的服务在您所在的国家/地区不可用

不可用截图

于是找百度搜索解决方案

  • 国内注册OpenAI账号

短信平台: https://sms-activate.org/

在注册是浏览器语言切换为英语

选择的是巴西的手机号作为验证。

账号注册好了

转载-注册流程

以下大部分内容来自Read dev Docs的文章。

前期准备

  • 科学上网，最好是美国IP

  • 外国邮箱，如gmail、outlook、iCloud邮箱等

  • 有一个能收到验证码的外国手机号码，如果你没有，请跳到下一步，注册一个能接验证码的手机号，注意这是收费的！

注册短信平台

打开 https://sms-activate.org/ (opens new window)点击右上角注册

使用邮箱和密码进行注册，激活账号后点击充值，使用支付宝充值1美元

输入1美元支付

支付了可能要等几分钟才显示到余额账户。

注册 OpenAI 账号

挂上科学访问打开 https://beta.openai.com/signup (opens new window)进行注册

注意：将浏览器语言修改为英语

注册成功后会进入验证手机号页面，选择你下一步短信平台的接码的手机国家

在短信平台接码

右侧服务，输入搜索openai，选择

选择印度或者巴西

现在你有一个可以接码的号码了，复制号码到OpenAI验证码界面

输入接码号码，点击send code

如果没有收到短信验证码，务必在有效期(20分钟内)，点击x免费退一次，重新选择购买一次。

注册完就可以去登录 https://chat.openai.com/auth/login使用
